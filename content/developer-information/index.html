Cast issue tracker: [https://github.com/cloudkick/cast/issues](https://github.com/cloudkick/cast/issues ) Website issue tracker: [https://github.com/cloudkick/cast-site/issues](https://github.com/cloudkick/cast-site/issues ) Vagrant image issue tracker: [https://github.com/cloudkick/cast-site/issues](https://github.com/cloudkick/cast-site/issues ) ### Continuous Integration Cast continuous integration server powered by Buildbot can be found at [http://buildbot.cast-project.org](http://buildbot.cast-project.org) ### Mailing list While the main development and code reviews happen on Github we use mailing list to discus about Cast roadmap, features and other project related things.
