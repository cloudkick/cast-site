{% extends "skeleton/_body_md.html" %} {%hyde title: "JSlint" navname: "JSlint" created: 2011-03-24 %} {% block content_md %} To enforce some of our style conventions, we use JSLint. JSlint has been tweak and modified according to our needs (we have disabled some checks and added support for Node global objects). To run JSlint on all the source files, run the following command in the Cast root directory:

{% syntax console %} vagrant@vagrantup:/data/cast$ scons jslint scons: Reading SConscript files ... scons: done reading SConscript files. scons: Building targets ... {% syntax console %} scons jslint files="lib/utils/command_parser.js lib/util/misc.js" {% endsyntax %} {% endblock %}
