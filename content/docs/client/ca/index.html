{% extends "skeleton/_body.html" %} {%hyde title: "Cast Client - Certificate Signing Request Management" created: 2011-03-01 listing: True %} {% block content_body %} {% markdown %} Cast offers x509 certificate based authentication of clients. But signing, distributing and managing certificates can be pain, especially if you don't already have infrastructure in place to do so. To alleviate this, cast includes basic certificate management functionality accessible through the "ca" command. When you add a remote, cast will generate a Certificate Signing Request and send it to the newly added remote agent. Then, either from the server on which the agent resides, or from any other approved client the request can be approved for signing.
