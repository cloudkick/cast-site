Cast Client - Adding a Remote

Add a reference to a remote cast-agent

Synopsis

cast remotes add [OPTIONS] NAME URL

Required Arguments

NAME

A name for the remote

URL

The URL to the remote

Options

--default, -d

Make this remote the default. Up to one default remote may be specified for each project, as well as one global default. If another default already exists, this will become the default instead.

--overwrite, -o

Use this option if you want to overwrite an existing remote.

--project, -p

Add the remote only to the current project. This will override any global remote with the same name, and a default project remote, if one exists, will take precedent over the global one.

--accept, -y

Accept the remote certificate without prompting. This is almost certainly a bad idea.
