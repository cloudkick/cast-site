Cast Client - Creating a Project Manifest

Create a simple manifest file by answering a series of questions. Optionally specify manifest parameters from the command line.

Synopsis

cast init [OPTIONS] ENTRYFILE

Required Arguments

ENTRYFILE

The entry file of the application

Options

--apppath <path>, -a <path>

Path to the root of the application

--name <name>, -n <name>

The name of the application.

--type <type>, -t <type>

Specify the type of the applicaton. Available types are 'nodejs' and 'shell'

--template <file>

Specify files that should be rendered as templates during deployment. Use once for each file to be templated.

--datafile <file>, -d <file>

Specify files that may be programatically modified during operation of the application. These will not be replaced during upgrades

--force, -f

Replace an existing manifest file.
