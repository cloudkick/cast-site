Cast Client - Uploading Bundles

Upload an application bundle to a remote server.

Synopsis

cast bundles upload [OPTIONS] [VERSION]

Optional Arguments

VERSION

The bundle version to upload, defaults to the newest available

Options

--remote <remote>, -r <remote>

The remote to use for this command
